# Adabet frontend


## Quick start

- Install modules

	```bash
	cp .env.sample .env
	
	npm install
	```

- Run

	```bash
	# Build as dev (fast build, poor perfomance)
	npm run dev

	# Build as build (slow build, good performance)
	npm run build

	# Start server
	npm start
	```


## Tips

- Lack modules

	```bash
  # Install `sharp`
	npm install sharp --save --force
	```
